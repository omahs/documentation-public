---
title: "Orderbook"

---

`{symbol}@orderbook` depth 100 push every 1s


**Parameters**

| Name          | Type   | Required | Description           |
|---------------|--------|----------|-----------------------|
| id            | string | Y        | id generate by client |
| event         | string | Y        | subscribe/unsubscribe |
| topic         | string | Y        | `{symbol}@orderbook`    |
| params.symbol | string | Y        | `{symbol}`              |
