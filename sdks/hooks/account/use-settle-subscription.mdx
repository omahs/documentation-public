---
title: "useSettleSubscription"
---

Subscribe to [PnL settlements](/build-on-evm/evm-api/websocket-api/private/pnl-settlement) for current account via Websockets.

- [Tech docs](/sdks/tech-doc/modules/orderly_network_hooks#usesettlesubscription)

### Example

```ts
useSettleSubscription({
  onMessage: (data: any) => {
    const { status } = data;

    switch (status) {
      case "COMPLETED":
        // settlement completed
        break;
      case "FAILED":
        // settlement failed
        break;
      default:
        break;
    }
  }
});
```
